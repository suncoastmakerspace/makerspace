Project Summary:

GraphingCalculator is a Java-based desktop app built using Swing that allows users to input math expressions and visualize them as graphs in real-time.
This Java application focuses on simplicity and clarity, maintaining a clean internal structure.

Requirements:

To run this project, you will need:
- Java Development Kit (JDK) 8 or higher (too new not supported)
- A Java-compatible IDE (optional, as long as you have the condition)
  
No external libraries are required, the project uses only standard Java and Swing.
